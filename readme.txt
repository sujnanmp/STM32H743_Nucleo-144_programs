IN THE VIRTUAL BOARD CLEAR ALL THE PINOUTS
ADD THE GPIO OUTPUT AND INPUTS TO THE PINS U WANT TO USE (HERE WE USE PIN 14 AND PIN 1)
SYSTEM CORE > RCC > HIGH SPEED CLOCK > CRYSTAL CERAMIC RESONATOR 
SET THE DEBUG CONFIGURATIONS 
CONNECT AND CONFIGURE THE BOARD 


FOR ALTERNATIVE BLNKING
User LD1: A green user LED is connected to the STM32H7 I/O PB0 (SB39 ON and SB47
OFF) or PA5 (SB47 ON and SB39 OFF) corresponding to the ST Zio D13.
User LD2: A yellow user LED is connected to PE1.
User LD3: A red user LED is connected to PB14
